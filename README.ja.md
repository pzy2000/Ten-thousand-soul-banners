# Ten Thousand Soul Banners.Skill

> 公開情報にもとづき、著名人・ネット人物・架空キャラクターを再利用可能な `.skill` モジュールへ蒸留するリポジトリです。

## 言語
[中文](README.md) | [English](README.en.md) | [Español](README.es.md) | **日本語** | [한국어](README.ko.md)

## このリポジトリについて
複数ペルソナを扱う Skill モノレポです。

現在の構成:
- 7 ペルソナ
- 1 つのルーター skill（`all-personas`）
- 5 つのカテゴリページ
- 今後の追加に使えるテンプレート

## クイックインストール
```bash
git clone https://github.com/pzy2000/Ten-thousand-soul-banners.git
cd Ten-thousand-soul-banners
cp -R skills/all-personas ~/.codex/skills/all-personas
```

## 誠実な境界
このプロジェクトができること:
- 公開素材から安定した話し方・判断フレームを再現する
- ペルソナごとに異なる分析視点を提供する

このプロジェクトができないこと:
- 本人の代替
- 常に最新事実の保証
- なりすまし・詐欺・欺瞞目的での利用
