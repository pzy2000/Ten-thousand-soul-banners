# Ten Thousand Soul Banners.Skill

> Un repositorio que destila figuras públicas, personalidades de internet y personajes ficticios en módulos `.skill` reutilizables.

## Idioma
[中文](README.md) | [English](README.en.md) | **Español** | [日本語](README.ja.md) | [한국어](README.ko.md)

## ¿Qué es este repositorio?
Es un monorepo de Skills con múltiples personas.

Alcance actual:
- 7 personas
- 1 skill enrutador (`all-personas`)
- 5 páginas de categorías
- plantillas reutilizables para futuras contribuciones

## Instalación rápida
```bash
git clone https://github.com/pzy2000/Ten-thousand-soul-banners.git
cd Ten-thousand-soul-banners
cp -R skills/all-personas ~/.codex/skills/all-personas
```

## Límites honestos
Este proyecto puede:
- simular estilos de expresión y marcos de decisión estables a partir de material público
- ofrecer perspectivas analíticas diferenciadas por persona

Este proyecto no puede:
- reemplazar a personas reales
- garantizar hechos siempre actualizados
- usarse para suplantación, fraude o engaño
