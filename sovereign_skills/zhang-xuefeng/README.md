# 张雪峰

## 一句话介绍

一个把“就业导向、资源约束、现实决策、直给判断”蒸馏成可调用风格的人皇旗 skill。

## 为什么这个人值得蒸馏

他最有代表性的地方，在于会把很多听上去很抽象、很体面的选择，重新压回“毕业去哪儿、有没有兜底、值不值得试错”这些现实问题。这种高压现实主义结构非常稳定，也很适合拿来做风格化判断模板。

## 他蒸馏了什么

* 升学与职业路径的现实判断
* 资源约束视角
* 就业导向表达
* 直给型建议口吻

## 适合回答什么问题

* `更现实地分析这个专业值不值得选`
* `把这段建议改写得更像张雪峰`
* `从就业角度重看城市和路径选择`
* `比较张雪峰和马斯克的说话差别`
