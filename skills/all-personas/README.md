# all-personas

## 一句话介绍

这是“万魂幡.Skill”的总入口 skill，用来列人、分流、切换 persona，并在需要时安排多角色对照。

## 为什么这个总 skill 值得存在

多人 monorepo 如果没有统一入口，用户就只能靠翻目录选角色。`all-personas` 解决的是“怎么找到对的人格模块”，而不是“把所有人格胡乱拼在一起”。

## 目录位置

* 总入口保留在 `skills/all-personas/`
* `renhuang-flag` 角色位于根目录 `sovereign_skills/`
* 其余角色位于根目录 `soulbanner_skills/`

## 它蒸馏了什么

* 角色索引
* 分类浏览
* 路由切换
* 问题到 persona 的匹配建议
* 多角色对照入口

## 适合回答什么问题

* `列出万魂幡里所有角色`
* `谁适合分析这个问题`
* `切到常熟阿诺模式`
* `用人皇旗的视角看这件事`
* `让特朗普和马斯克分别说一句`

## 触发词

* `列出万魂幡里所有角色`
* `切到某某模式`
* `谁适合分析这个问题`
* `用人皇旗的视角看这件事`

## 使用规则

* 默认一次只激活一个主角色
* 默认不自动人格融合
* 多角色模式必须显式说明谁在说话
* 分类用于浏览，不是现实评价体系

## 诚实边界

* 多角色模式只是并列输出，不表示人格融合成功
* 分类命名是戏谑标签，不代表现实价值判断
