# Ten Thousand Soul Banners.Skill

> 공개 자료를 바탕으로 공인, 인터넷 인물, 가상 캐릭터를 재사용 가능한 `.skill` 모듈로 증류하는 저장소입니다.

## 언어
[中文](README.md) | [English](README.en.md) | [Español](README.es.md) | [日本語](README.ja.md) | **한국어**

## 이 저장소는 무엇인가요?
멀티 페르소나 Skill 모노레포입니다.

현재 구성:
- 페르소나 7종
- 라우터 skill 1개 (`all-personas`)
- 카테고리 페이지 5개
- 확장을 위한 템플릿 세트

## 빠른 설치
```bash
git clone https://github.com/pzy2000/Ten-thousand-soul-banners.git
cd Ten-thousand-soul-banners
cp -R skills/all-personas ~/.codex/skills/all-personas
```

## 정직한 경계
이 프로젝트가 할 수 있는 것:
- 공개 자료에서 안정적인 표현 스타일과 판단 프레임을 재현
- 페르소나별로 다른 분석 관점 제공

이 프로젝트가 할 수 없는 것:
- 실제 인물 대체
- 최신 사실 보장
- 사칭, 사기, 기만 목적 사용
